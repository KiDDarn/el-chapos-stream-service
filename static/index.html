📖 How to stream from VLC Player [Click to Expand]

You can stream high-quality files directly from your VLC media player onto the server using this shell command:

vlc your-media-file.mp4 --sout="#transcode{vcodec=h264,vb=2000,acodec=mpga,ab=128,channels=2,samplerate=44100}:std{access=livehttp{seglen=4,delsegs=true,numsegs=5,index=/home/m4p3/webrtc-stream-app/static/hls/stream.m3u8,index-url=/hls/stream-########.ts},mux=ts,dst=/home/m4p3/webrtc-stream-app/static/hls/stream-########.ts}"

Note: Once VLC starts streaming, viewers will get a button to switch automatically to the high-quality VLC feed.

🎙️ How to share System Audio on Linux [Click to Expand]

To capture and stream all desktop audio (including application sounds) into your stream on Linux:

  1. Run this command in a terminal to create a virtual input device monitoring your speaker/headphones output: 
    pactl load-module module-remap-source master=$(pactl get-default-sink).monitor source_name=virtual_system_audio source_properties=device.description="System_Audio_Mix"
  2. In the dashboard, set Capture Source to Web Camera & Microphone, and click Share Stream.
  3. When the browser requests camera/microphone permissions, choose "System_Audio_Mix" as your microphone source.
